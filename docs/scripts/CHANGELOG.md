# Changelog

## v1.2.0 (2026-07-19)
- Renamed app to "Signal Pro"
- Added multi-instrument support (15+ symbols)
- Added Alpha Vantage as second API provider
- Added per-symbol decimal precision (Forex=5, JPY=3, Gold=2)
- Added configurable auto-refresh intervals
- Added candlestick thickness setting
- Added alert sound selection
- Fixed dynamic symbol display in PriceCard
- Fixed local timezone display

## v1.1.0 (2026-07-19)
- Added SymbolSelector widget
- Added symbol field to PriceAlert model
- Updated all screens to support symbol switching

## v1.0.0 (2026-07-18)
- Initial release as "Gold Signal AI"
- XAU/USD only with signal engine
- EMA, RSI, MACD, Bollinger Bands, ATR indicators
- Price alerts with notifications
- AdMob integration
- GitHub Actions CI/CD for APK builds
